This Project is meant to provide the theatre sitting in the seats available under each section of the sitting layout.

**The Project has to be opened in Netbeans or any JRE supported IDE, where we can build and run this application.
Contents of the project such as dataFile.txt contains the input data required for the logic to perform.

> Exception has been handled in the main method.
> The Java class is created to seperate logic from the main method.
> Parsing has been done in Main method.

**Please run the code and feel free to test other use cases.
